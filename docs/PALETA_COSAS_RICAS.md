# Paleta de colores · Cosas Ricas

Extraída de la identidad visual (logo, Viernes de Match).

## Colores principales

| Nombre | Hex | Uso |
|--------|-----|-----|
| **Rojo Cosas Ricas** | `#E04E4A` | Acento, botones, links |
| **Crema** | `#F8F0E3` | Fondos cálidos |
| **Blanco** | `#FFFFFF` | Texto sobre rojo, iconos |

## Variaciones

| Nombre | Hex | Uso |
|--------|-----|-----|
| Rojo oscuro | `#C94542` | Hover, estados activos |
| Rojo claro | `#E86A66` | Hover suave |
| Crema clara | `#FDF8F3` | Fondos muy suaves |
| Crema oscura | `#EDE4D5` | Bordes, separadores |

## En CSS

```css
--color-brand-red: #E04E4A;
--color-brand-cream: #F8F0E3;
--color-brand-white: #FFFFFF;
--color-accent: #E04E4A;  /* Integrado en design-system */
```
